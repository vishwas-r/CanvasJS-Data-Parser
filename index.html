CanvasJS Chart DataParser

This plugin allows you to parse custom data-format to format accepted by CanvasJS. Check out Github Repository for more info / examples.

Live Examples

1. Datapoints as Array

How to Use?

var chart = new CanvasJS.Chart("chartContainer", {
	.
	.
	.
	//Chart Options
	data: [{
		dataPoints: CanvasJS.parseData([71,55,50,65,95,68,28,34,14])
	}]
	.
	.
	.
	.
});
chart.render();

Datapoints from JSON

How to Use?

var chart = new CanvasJS.Chart("chartContainer", {
	.
	.
	.
	//Chart Options
	data: [{
		type: "column"
	}, {
		type: "column"
	}]
	.
	.
	.
	.
});
chart.parseJSONAndRender("https://api.npoint.io/6f8153fa498ba8522ab8", {x: "date", y: "units"});
